# ui-design

UI 设计相关 Skill 合集，收录排版、配色、动效、交互、响应式等领域的实用工具。

## 基础

| Skill | 说明 | 来源 |
|---|---|---|
| [frontend-design](frontend-design/SKILL.md) | 前端设计核心 Skill，包含排版、配色、空间设计、动效、交互、响应式、UX 文案 7 个维度的参考文档 | [pbakaus/impeccable](https://github.com/pbakaus/impeccable) |
| [teach-impeccable](teach-impeccable/SKILL.md) | 收集当前项目的设计上下文，写入配置，作为后续所有命令的基准 | [pbakaus/impeccable](https://github.com/pbakaus/impeccable) |
